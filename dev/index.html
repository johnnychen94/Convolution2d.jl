Examples

using Convolution2D
using Images, TestImages

camera = imresize(testimage("cameraman"), 255, 255)
kern = [0 1 0
		1 -4 1
		0 1 0]

X = float.(camera)
conv2D(X, kern)
imfilter(X, kern)

Convolution2D.conv2D — Function
conv2D(X, kern)

Apply 2D convolution kernel kern on array X.

Arguments

  • X::AbstractArray: must be odd size.
  • kern::AbstractArray: usually small array, must be odd size.
